## 3. 기본 세팅
`MySQL Connection` 에서 들어갈 비밀번호를 설정한다.

## 4. 사용할 database 생성

```mysql
create Database [데이타베이스 이름] 
```
참고로 대소문자 구분 없다 

## 5. Use
```mysql
use [사용할 데이터 베이스 이름]
```
사용할 데이터베이스를 먼저 정해야 작동할 수 있다
